# Advertising Center — стан v15

Контрольоване злиття старого рекламного модуля з чинною фермою завершено.

Підключено:

- публічну форму `/advertise` для заявок рекламодавців;
- таблиці заявок і рекламних подій у SQLite;
- кампанії `affiliate`, `sponsor`, `own_product`;
- моделі виплат `CPS`, `CPA`, `CPL`, `FLAT`;
- tracking slug і UTM;
- click/conversion/revenue/refund та EPC;
- адмінські API для звіту, подій і статусів;
- сумісність із чинними `/go/<offer_id>` та `/r/<offer_id>`.

Перевірено: створення кампанії, click, conversion, summary і advertiser lead.
Робочі affiliate-маршрути не переписувалися поверх старої версії.

Гроші не гарантуються кодом: потрібні реальні офери, трафік і підтверджені
конверсії. Кампанії без економіки ставляться на паузу через API статусу.
