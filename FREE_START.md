#### 1.2 Pexels API (безкоштовні відео)

```
1. Перейдіть: https://www.pexels.com/api/
2. Sign Up
3. Отримайте API key (миттєво)
```

**Ліміти:** Безлімітно для некомерційного використання

#### 1.3 YouTube API (публікація)

```
1. https://console.cloud.google.com/
2. New Project → "YouTube Automation"
3. Enable "YouTube Data API v3"
4. Credentials → OAuth 2.0 Client ID
5. Download credentials
```

**Ліміт:** 10,000 units/день (достатньо для 6 відео/день)
