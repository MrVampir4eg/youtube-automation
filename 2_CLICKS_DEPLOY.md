## 🚀 КРОК 2: Завантажити проект (3 хв)

### У Render Dashboard:

```
1. Натисніть "New +"

2. Виберіть "Web Service"

3. Натисніть "Build and deploy from a Git repository"

4. ВНИЗУ натисніть "Public Git repository"

5. Repository URL: 
   Тут можна вставити будь-який Git URL
   
   АБО натисніть "Deploy without connecting Git"
```

### Налаштування:

```
Name: youtube-automation

Runtime: Python 3

Build Command: 
pip install -r requirements-free.txt

Start Command:
python -m dashboard.app

Instance Type: Free
```

### Environment Variables:

**НАТИСНІТЬ "Add Environment Variable" і додайте ЦІ:**

```
GROQ_API_KEY=YOUR_GROQ_API_KEY

PEXELS_API_KEY=ВАШ_PEXELS_API_KEY

USE_FREE_MODE=True

FREE_CONTENT_GENERATOR=groq

FREE_VOICE_SYNTHESIZER=gtts

FLASK_PORT=5000

FLASK_DEBUG=False

DATABASE_URL=sqlite:///youtube_automation.db

TIMEZONE=Europe/Kiev
```
