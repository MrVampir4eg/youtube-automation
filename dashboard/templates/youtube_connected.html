Скопіюй refresh token нижче. Не надсилай його в чат і не показуй на скриншотах.

У Render → Environment додай:

YOUTUBE_REFRESH_TOKEN = скопійований token

AUTO_UPLOAD = True

YOUTUBE_PRIVACY_STATUS = public

Після Save Changes Render перезапустить сервіс. Token дасть змогу автоматично завантажувати нові відео.
